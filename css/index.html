CSS Basics

CSS stand for cascading style sheets. It is basically a set of rule and follow this pattern: selector { property: value; } . There are so many properties. Do not panic! You can find the information online and you do not need to remember them all.

How to Include Styles

Inline styles are not best practice. The best way to include styles is in a separate stylesheet. Use the link element to add the stylesheet. This is best as now our styles can be accessed by across our program.

Colors

Named colors: Most browser recognise around 140 colors. These are the most limited! Most monitors can display 16,000,000

R,G,B Colors

R,G,B is an additive color system. There are three channels ranging from 0 - 255. Here are some examples: rgb(255,0,0) rgb(0,255,0) rgb(0,0 255)

Hexidecimal

Hexidecimal is base 16, we use numbers 0-9 and letters A-F. Basically F is the same as writing 15, this is so we can store more value in a single digit. Just like RGB the first two are red, second two are green and the last two are blue!Hex example: #ff23f2

Pixels,Font Size and Font Family

We have many units to specify font size. Pixels are the most commonly used unit in absolute units. Not recommnded for responsive sites! font-size: 100px

Fonts are built into your browser. We can use custom fonts too.font family: verdana.

CSS Selectors

Universal selectors select everything, although it is not that useful in more complex projects. * {background-color: cyan}. The elements selector are more common and can be used to select more than one element when separated by a comma.h1, h2 {color: magenta}

Id and Class Selector

We can use the class and ID selector to style elements with a certain ID or class. The id is unique and there is only one per page used sparingly, there can be multiple elements with the same class.

Desendant Selectors

This will select all anchor tags nested inside the li li a {color: teal}.. For the example below we could add a class to the ul then style the li and the nested links like this: .descendant li a { text-decoration: none }.

Adjacent and Descendant Selectors

This will select elements that are on the same level. So for example a button directly after an input will be pink:
input + button {background-color: pink}.

Any element that is a direct descendant can be selected using the angle bracket although it is not so common.
div > li {color: white}

Attribute Selector

We can use square brackets to grab a selector by the type for example!
input[type="text"] { width: 300px; color: yellow;}

Psuedo Classes

Psuedo classes are modifiers used to show when an element has a different interaction such as hover. Psuedo classes start with a colon. To add a psuedo class to the button below I used this selector.
.hover:hover { background-color: #37bad6; color: #2b2954; }.

We can also add an active selector for when we click on a button.
.hover:active { background-color: pink; color: #2b2954; }

Nth of type can be used to apply styles to every other one, or the last one.
.box:nth-of-type(2n) { background-color: pink;}.

Psuedo Elements

This keyword lets you style a particular part of an element. For example the bullets in a list...

  • ::after
  • ::before
  • ::first-letter
  • ::first-line
  • ::selection

A final note...The order you declare your styles matters. If you try to add a color selector to a h1 for example twice the last selector will win.

WTF is Specificity?

What happens when there is a conflict in styles? What if we have a button with a class and just a regular button? The browser measures how specific a selector is and the most specific will win. This is the general formula to remember:
ID > CLASS > ELEMENT.
