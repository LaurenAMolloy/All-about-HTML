All About HTML

HTML, CSS and JavaScript

HTML stands for HyperTextMarkUp Language and is used to place the elements on a webpage. CSS stands for cascading style sheets and helps us to describe the elements. JavaScript is used to implement actions on our webpages. The ultimate goal of HTML is to give webpages some kind of structure and indcate to our browser how to format the text. HTML provides the rules to the browser. It was orginally designed to format academic papers.

Anchor Tags

The anchor tag is used to link us to another webpage or url. We must include the destination in the href attribute. An attribute is some information we can include in the tag. We can also link to another html file that we have created

Images

The img tag is used to add images to our web pages. It is a self closing tag. We add an attribute called src that provides the location of the image. If your image does not load you should provide an alt attribute!

The Form Element

The form element is a container that will hold form controls. We group the collection of inputs together using forms. The action attribute specifies where the form data should be sent. The method attribute specifies which HTTP method will be used.

The Input Element

The input element is a self closing element. The default type is text, but we should specify the type. The type will change how the input behaves.A placeholder can be used to indicate to the user what should entered. Labels give us information about and input but also associates the label to the input. Using label for="username" and giving an id attribute will connect the label and the input. The default behaviour for a button inside a form is submit!
